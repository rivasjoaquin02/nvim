# strangedevel - nvim

This config is based on the config from @ThePrimeagen

![screenshot](screenshot/image.png)

## Requirements
- ripgrep
- clang: lsp/formatter/linter for c/cpp
- ruff: lsp/formatter/linter for py
- tsserver: lsp for js/ts
